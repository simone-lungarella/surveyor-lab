Winforce

Spring Boot - Java - React - Javascript - MySQL

• Context

This React app helps users manage construction sites. They can create, update, and delete sites, and track progress by marking steps as complete.

There are two types of users: Admins, who can create users and have full access to the site list, and regular users. Authentication is handled through JWT tokens obtained from the backend upon providing the correct username and password. The app stores data using a PostgreSQL database hosted as a Heroku add-on.

• Backend

The backend is a Java microservice called event-handler and is responsible for handling CRUD operations on the construction sites and the steps. The API is reachable through the browser using swagger-ui by providing an existing username and password.

• Frontend

The frontend is a React application responsive and mobile friendly. It is deployed on a Github Pages. Is built with material-ui and offer a way to access and interact with each of the construction sites. Every profile created can have a fine-grain access to the sites with the possibility to interact only wwith a part of steps of each site.
