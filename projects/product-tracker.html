Product Tracker

Spring Boot - Java - React - Javascript

• Context

A React application that allows users to define production related information about their products from the cleaning phase of raw products to the final packaging of the product. The application make use of Formik and handle a set of six pages each of them containing information about a different phase of the production. Each form is finalized with a Download button that generates a PDF file with the information that the user has entered.

• Backend

The backend is a Java microservice called Foptility and is responsible for generating the PDF files. The microservice is deployed on a Heroku instance and exposes a REST API that is consumed by the frontend. This same microservize can be used from the browser by accessing the swagger-ui page. The code is open-source and can be found here.

• Frontend

The frontend is a React application responsive and mobile friendly. It is deployed on a Github Pages. The application make use of react-router-dom to handle the navigation between the different pages. Uses Formik and Yup to handle the forms and the validation of the inputs.
