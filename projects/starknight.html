Star Knight

Unity - C# - Aseprite

• Context

A top-down spaceship shooter game inspired by Space Invaders. LincoLn and I were responsible for the design and implementation which took place over a month from May 20th to June 10th, 2023.

• Concept

The game is a 2D platformer where you control a spaceship and the objective is to destroy waves of enemy spaceships. The player can move the spaceship using the arrow keys and shooting projectiles at the enemies using the space bar. The game ends if the player's spaceship is destroyed or after defeating all the enemy waves.

• Future Improvements

The game is a very small complete game that we made to improve our familiarity with Unity Engine. Even if it's complete it lacks some key feature and small improvements.

  • Make the game more accessible with different options that should be configurable.
  • Adding a more visible reload hint.
  • Only a playable level is currently available. A few more should be added.
  • There's no way yet to save the current state of the game. A save&load feature is needed.
  • There is only one type of enemy, which make the game monotonous.
  • A few power-ups could be added to make the level more satisfying to play
